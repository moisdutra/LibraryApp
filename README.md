# LibraryApp

Aplicação de gerenciamento de biblioteca em desenvolvimento, permitindo que usuários **reservem livros** e **efetuem devoluções** de forma intuitiva. Foco em controle de estoque, empréstimos e histórico de usuários.[web:11][web:18]

## Funcionalidades Planejadas
- Cadastro de livros (título, autor, disponibilidade).
- Perfil de usuários com histórico de reservas.
- Reserva e confirmação de empréstimo.
- Devolução com cálculo de multas (opcional).
- Dashboard para admins visualizarem status.[web:12][web:20]

## Tecnologias
- Java + Spring Boot (backend).
- Banco de dados relacional (H2 para testes).
- Frontend responsivo (em progresso).
- API REST para integrações.

## Como Executar
1. Clone o repo: `git clone https://github.com/moisdutra/LibraryApp.git`
2. `cd LibraryApp`
3. `mvn clean install`
4. `mvn spring-boot:run`
5. Abra [http://localhost:8080](http://localhost:8080).[web:12]
